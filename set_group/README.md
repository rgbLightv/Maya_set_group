# Set Group for Maya

Quick installation package for Set Group Maya plugin.

## Install

1. Drag-and-drop `install.py` into Maya viewport, or run in Script Editor:
   ```python
   exec(open(r"C:\path\to\install.py").read())
   ```

2. A shelf button **SG** will be created automatically.

3. Click **SG** to open the toolbar.

## Usage

- Select objects in order, click **+** to create a set
- Left-click set button to select in recorded order
- Right-click set button for more options

## Supported Versions

Maya 2018 / 2020-2023 / 2024-2025
